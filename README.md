> Clinic Landing Page is a web application designed to showcase a modern aesthetic clinic, highlighting its advanced procedures, qualified professionals, and personalized care. It serves as a comprehensive online presence to attract and inform potential clients.

## Techs

- React
- Vite
- TypeScript
- Tailwind CSS
- React Icons

## Installing it

To install it, follow these steps:

```bash
git clone https://github.com/dspedroborges/clinic-landing-page
cd clinic-landing-page
npm install
```

## Using it locally

To use it locally, follow these steps:

```bash
npm run dev
```
